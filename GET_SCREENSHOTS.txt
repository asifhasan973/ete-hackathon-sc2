This is the best way to get real screenshots of your actual pipeline running.

STEP 1: Install Docker Desktop
-------------------------------
1. Go to: https://www.docker.com/products/docker-desktop
2. Click "Download for Mac"
3. Choose:
   - Apple Silicon (M1/M2/M3): Download "Mac with Apple chip"
   - Intel Mac: Download "Mac with Intel chip"
4. Open the downloaded .dmg file
5. Drag Docker to Applications folder
6. Open Docker from Applications
7. Wait for Docker to start (you'll see a whale icon in your menu bar)

STEP 2: Start Jenkins
---------------------
Once Docker is running, open Terminal and run:

    cd "/Users/asifhasan/My Computer/Coding/test"
    chmod +x setup-jenkins.sh
    ./setup-jenkins.sh

This will:
- Start Jenkins in a Docker container
- Show you the admin password
- Tell you to go to http://localhost:8080

STEP 3: Setup Jenkins (First Time Only)
----------------------------------------
1. Open browser: http://localhost:8080
2. Copy the admin password from terminal
3. Paste it in Jenkins
4. Click "Install suggested plugins" (wait 2-3 minutes)
5. Create admin user:
   - Username: admin
   - Password: admin (or your choice)
   - Full name: Your Name
   - Email: your@email.com
6. Click "Save and Continue"
7. Click "Save and Finish"
8. Click "Start using Jenkins"

STEP 4: Create Pipeline Job
----------------------------
1. Click "New Item" (top left)
2. Enter name: "demo-cicd-pipeline"
3. Select "Pipeline"
4. Click OK

5. Scroll down to "Pipeline" section
6. In "Definition" dropdown: Select "Pipeline script"
7. In the script box, paste the contents of your Jenkinsfile:

   Open another terminal tab and run:
   cat "/Users/asifhasan/My Computer/Coding/test/Jenkinsfile"
   
   Copy ALL the text and paste into Jenkins

8. Click "Save"

STEP 5: Run Pipeline and Take Screenshots
-----------------------------------------
1. Click "Build Now" (left sidebar)
2. Watch the pipeline run - it will show stages in the main area
3. Wait for it to complete (may fail at Docker stages, that's OK for screenshots)
